Comandaos De Formatação De Texto Do HTML4 Ainda Aceitos No HTML5

Apartir do HTML5 os comandos de formatação de texto e stilo, como cor, tamano e fonte, passou a ser comandado pelo CSS. Mas ainda o HTML5 aceita alguns comandos de formação simples. Como o negrito e itálico.

Exemplos
Negrito     <b>   </b>
Itálico     <i>   </i>
enfase     <em>   </em> (esse comando enfatiza para o navegador entender que essa palavra é muito importante).
Texto Deletado     <del>   </del>

Comandos de formatação em CSS

Apartir do HTML5, o CSS é responsavel pela formatação e stilo da pagina.

Exemplo

Sublinhado <spam style="text-decoration: underline;"> <spam>
Linha a Cima do Texto <spam style="text-decoration: overline;"> <spam>
Riscado <spam style="text-decoration: line-through;"> <spam>
Isolamento da formatação <spam style="text-decoration: none;"> <spam>


Negrito Nivel Normal <spam style="font-weight:normal;"> <spam>
Negrito Nivel Bold <spam style="font-weight:bold;"> <spam>
Negrito Nivel Bolder <spam style="font-weight:bolder;"> <spam>
Negrito Nivel 450 <spam style="font-weight:450;"> <spam>
Negrito Nivel 900 <spam style="font-weight:900;"> <spam> (A Variação é entre 100 até 900)

H2O Texto Sobrescrito <sup> </sup>
H2O Texto Subscrito <sub> </sub>
